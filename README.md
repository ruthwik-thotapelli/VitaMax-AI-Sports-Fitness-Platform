# 🚀 VitaMax – AI Sports & Fitness Platform

AI-powered full-stack Sports & Fitness platform built with **React.js, Tailwind CSS, PHP Laravel, and MySQL**.
VitaMax delivers a modern startup-level fitness ecosystem with OTT-style workout experiences, AI-based recommendations, analytics dashboards, nutrition tracking, trainer management, and community-driven fitness engagement.

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/ruthwik-thotapelli/VitaMax-AI-Sports-Fitness-Platform.git
```

---

# 🔥 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

# 🔥 Backend Setup

```bash
cd backend
composer install
php artisan migrate
php artisan serve
```

Backend runs on:

```bash
http://127.0.0.1:8000
```

---

# 🗄️ Database Setup

Create MySQL database:

```sql
CREATE DATABASE vitamax;
```

Update `.env` file:

```env
DB_DATABASE=vitamax
DB_USERNAME=root
DB_PASSWORD=
```
